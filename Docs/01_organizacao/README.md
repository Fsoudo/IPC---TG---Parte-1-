# Secção 1 - Ferramentas e Organização da Equipa

## Objetivo
Configurar as ferramentas de gestão e organização do projeto EasyMed.

## Ferramenta Escolhida
**Trello** - board com 3 colunas: `Épicos`, `Tarefas` e `Sprints`.

| Coluna | O que contém |
|--------|-------------|
| `Épicos` | Grandes áreas do projeto (cada secção do trabalho) |
| `Tarefas` | Trabalho concreto dentro de cada épico, atribuído a um elemento |
| `Sprints` | Tarefas planeadas para o período de trabalho atual |
